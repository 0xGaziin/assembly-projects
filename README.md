# 🛠️ Assembly Lab: Luma OS Edition

A collection of low-level projects and experiments written in x86-64 Assembly.

## 📖 About This Repository

This is a personal sandbox dedicated to practicing Assembly language. The goal is to understand how software interacts directly with the kernel and CPU registers without the abstraction layers of high-level languages.

Each project focuses on a specific concept, such as:

    System Calls: Interacting with the Luma kernel (Reading/Writing).

    Memory Management: Working with .data, .bss, and Stack manipulation.

    Control Flow: Conditional jumps, loops, and logic comparisons.

    Data Conversion: Handling ASCII strings vs. raw integers.

## 🚀 Environment & Tools

    OS: Luma OS 1.0 New Moon

    Architecture: x86-64 (AMD64)

    Assembler: NASM

    Linker: ld (GNU Linker)
